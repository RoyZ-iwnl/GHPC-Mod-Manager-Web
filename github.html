GitHub 访问说明

⚠️ 网络环境提示
由于大陆网络环境限制，GitHub 可能无法直接访问或访问缓慢。
🔧 解决方案
为了正常使用本程序的下载相关功能，建议采用以下方式：
方法一：使用 VPN 或梯子工具
推荐使用稳定的 VPN 服务来访问 GitHub，这是最直接有效的方式。
方法二：使用反代工具
可以使用 steamcommunity302 等反代工具来改善访问。
参考教程：https://www.dogfight360.com/blog/18682/
方法三：程序内置代理加速
您也可以稍后在程序设置中开启 GitHub 访问代理加速功能。
注意：内置代理速度可能不够理想，建议优先考虑前两种方法。
如果您已经配置好网络环境，可以关闭此页面继续使用程序。
